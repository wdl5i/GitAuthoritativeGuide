# 第一章 git初使化
## 创建版本库及第一次提交
    1. 配置当前用户名和邮箱地址：
    git config --global user.name "xxx"
    git config --globel user.email "xxx@xxx.com"
    2. 设置git别名
    sudo git config --system alias.ci commit
    或 git config --global alias.ci commit
    3. 命令行中开户颜色
    git config --global color.ui true
